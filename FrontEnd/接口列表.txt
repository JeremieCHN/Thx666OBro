登录，注册
 ├ url : 你决定吧
 ├ 方式: Post
 ├ 数据: username:手机号,password:密码（暂时先用明文吧）,checkword:验证码
 ├ 返回: 字符串，username:用户名;status:true/false   // 要不要在返回值里带上失败原因
 └ PS  : 全程没有双引号，密码限制为只能用数字和字母，所以不会和其他字符混在一起

手机号查重
 ├ url : 你决定吧
 ├ 方式: Get
 ├ 数据: ?num=手机号
 └ 返回: status:true/false   // 要不要在返回值里带上失败原因


请求发送验证码
 ├ url : 你决定吧
 ├ 方式: Get
 ├ 数据: ?username=手机号       // QueryString
 └ 返回: time:时间戳            // 有效期，设为0表示发送失败

 ├ 方式: Get
